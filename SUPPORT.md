## Bugs and Questions

Use GitHub Issues for reproducible bugs, documentation problems, and general
questions. Include:

- CyberTabletop commit or release tag,
- install path,
- Docker version,
- browser,
- steps to reproduce,
- relevant logs with secrets removed.

Do not post passwords, invite codes, API keys, cookies, real incident data,
private logs, database dumps, or regulated data.

## Security Reports

Do not report suspected vulnerabilities in a public issue. Follow
[SECURITY.md](SECURITY.md).

Security-sensitive topics include authentication, authorization, MFA, session
handling, audit logging, Docker/network exposure, TLS/security headers, SSRF,
dependency vulnerabilities, AI provider connectivity, and data protection.
